# Plugin System

## Overview
A.I.M. hooks into the OpenCode runtime via a TypeScript plugin using the `@opencode-ai/plugin` SDK.

## Plugin Location
`.opencode/plugins/aim-hooks.ts`

## Hook Types

### `session.idle`
Triggers the session summarizer when the user becomes idle. Background task that writes session summaries for continuity.

### `tool.execute.after`
Writes a cognitive mantra counter to `continuity/MANTRA_PULSE.md` after every tool execution. Serves as a heartbeat/activity log.

### `experimental.session.compacting`
Injects AIM continuity context during session compaction events. Ensures the reincarnated agent inherits epistemic certainty.

## Installation
`install_opencode_plugins()` in `aim_init.py` handles plugin registration.

## Validation
`ensure_hooks_mapped()` in `aim_cli.py` checks `.opencode/plugins/` directory first before falling back to legacy hook paths.
