# GitOps Pattern

## Branch Strategy
- **Active work:** `opencode` branch
- **Never commit directly to `main`**
- All commits pushed to `origin/opencode`

## Remote
- Origin: `d3c12yp7012/aim-opencode`
- Forked from: `BrianV1981/aim`

## Commit Rules
1. **Surgical staging only** — `git add <specific-files>`, never `git add .`
2. Work pushed after each completed phase
3. Pre-commit hooks run via `./pre-commit.sh`

## Hygiene
- `workspace/` directory listed in `.opencodeignore` to prevent recursive scanning of git worktrees
- Git worktrees cleaned up after issues are resolved
