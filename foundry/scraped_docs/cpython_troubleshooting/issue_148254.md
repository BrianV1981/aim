## The Solution / Discussion
### Response 1
We can always say "sec", we use singular for the other units:

```console
❯ ./python.exe -m timeit --verbose "import time, random; time.sleep(random.random() * 5)"
1 loop -> 3.01 secs

raw times: 110 msec, 4.57 sec, 2.07 sec, 1.07 sec, 4.21 sec

1 loop, best of 5: 110 msec per loop

:0: UserWarning: The test results are likely unreliable. The worst time (4.57 sec) was more than four times slower than the best time (110 msec).
```
